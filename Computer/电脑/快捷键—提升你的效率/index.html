1、基本操作篇

新建快捷键 ——【Ctrl+N】
打开快捷键 ——【Ctrl+O】
撤销快捷键 ——【Ctrl+Z】
恢复快捷键 ——【Ctrl+Y】
查找快捷键 ——【Ctrl+F】
剪切快捷键 ——【Ctrl+X】
复制快捷键 ——【Ctrl+C】
粘贴快捷键 ——【Ctrl+V】
全选快捷键 ——【Ctrl+A】

以上都是基本操作，能够通用于大部分软件

收起所有窗口 ——【Win+M】
放大 / 缩小 ——【win+ +/-】
我的电脑 ——【win+E】
桌面 ——【win+D】
切换窗口 ——【win+Tab】
锁屏 ——【win+L】
新建文件夹 ——【ctrl+shift+N】
分屏 ——【win + 移动键】
中英输入法切换 ——【shift】
切换输入法 ——【ctrl+shift】/【Ctrl + 空格】

2、触摸板操作篇

单击：一个手指点击触摸板
双击：一个手指点两下触摸板
右击：两个手指同时点下
拖曳：一个手指点两下后按住不动，再拖动
切换程序：三指右滑，左滑
回到桌面：三指下滑
滚轮：双指上下移动

3、强制退出篇

1. 可以按 Alt+F4，这个一般是直接关窗口
2. 可以按 Ctrl+Esc 退回桌面
3. 可以按 Alt+Tab 键切换运行窗口，切出来再用鼠标右键选菜单关闭或者任务管理器关闭
4. 可以按 Ctrl+shift+esc 退出
5. 可以按 Ctrl+Alt+End 退出
6. 可以按 Ctrl+Alt+Del 进入任务管理器
7.ALT+F4 再按 U 可直接关机，再按 R 可重启

5、输入法智能输入篇

1. 人名：【姓名拼音 +；+r】——> 取消：【；】
2. 生僻字：【u + 拆解字拼音】
3. 金额变大写【v + 金额】

6、浏览器篇

关闭当前窗口 ——【ctrl+W】——> 恢复 ——【ctrl+shift+T】
关闭所有窗口 ——【ctrl+shift+W】
Ctrl+N ——【打开新窗口】
Ctrl+T ——【打开新标签页】
Ctrl+Shift+N ——【在隐身模式下打开新窗口】

7、其他篇
