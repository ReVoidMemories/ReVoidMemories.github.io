Vimium— 解放你的鼠标

首页 计算机 电脑

一个插件解放你的鼠标——Vimium插件，本插件快捷键区分大小写（以Chrome Google浏览器为例，其他浏览器自测）

大写快捷键我们可以使用 shift + 字母来实现，如果你打开了键盘上的大写锁定（caps lock）可以忽略。当然有些网页不一定能够使用或者功能会有所差异，请勿见怪。

如嫌麻烦可以跳转到精简版

# 实现页面的上下滚动

向上滚动 ——j（慢）/d（快）

向下滚动 ——k（慢）/u（快）

如果我们需要页面顶端或者页面底端我们可以使用
gg 回到页面顶端

G 回到页面底端

当然我们也可以使用其他的快捷键

键盘的上下键、pgup、pgdn 也可以达到相同的效果

# 标签页

当我们需要创建新标签页时我们可以使用 t 来实现
（你也可以尝试一下其他方法 ctrl+N）

当我们打开了太多的标签页时，
我们只需要按 T 来搜索你想打开的标签页
（缺点就是显示的标签页不能翻动）

快速切换至左右标签页也是很实用的
J—— 转到左边标签页
K—— 转到右边标签页

其他方法：
ctrl+shift+tab—— 转到左边标签页
ctrl+tab—— 转到右边标签页

标签页的关闭与恢复

x—— 关闭（我们可以不需要再用鼠标到页面上方点 × 了）
X—— 恢复上一个关闭的标签页

其他方法：
Ctrl+W（关闭）/ Ctrl+shift+T（恢复）

跳转到第一个标签页 ——g0（是 “零” 不是 “哦”）

其他方法：
ctrl+1，同理我们也可以 ctrl+2,ctrl+3… 来转到相应标签页
（太多标签页我也没办法，其实我认为下面这个方法更好用，因人而异吧，你喜欢就好。）

# 其他

yy—— 复制当前网页网址，
如此我们就不需要再 Ctrl+L（转到搜索栏）；Ctrl+C（复制）
&
直接复制

刷新页面 ——r
其他方法：ctrl+r、F5

复制当前标签页到新标签页 ——yt

转到第一个输入框 ——gi（只能转到第一个有点遗憾呀）

Alt + 左右键 —— 前进后退（没安装也有此功能哦！）

搜索并打开书签
当前页 ——b
新标签页 ——B

还可以打开 URL、书签、历史记录
o 表示在当前页打开
O 表示在新标签页打开

打开页面内链接
f 在当前页中打开
F 在新标签页中打开

当你按下 f/F 时（仿佛打开了新世界的大门）

# 常规操作

快捷键操作快捷键操作
j，<c-e>向下滚动k，<c-y>向上滚动
gg滚动到页面顶部G滚动到页面底部
d向下滚动半页u向上滚动半页
h向左滚动l向右滚动
r重新加载页面
yy将当前 URL 复制到剪贴板
p在当前选项卡中打开剪贴板的 URLP在新标签页中打开剪贴板的 URL
i我进入插入模式v进入可视模式
gi将页面上的第一个文本输入集中
f在当前选项卡中打开一个链接F在新标签页中打开链接
gf选择页面上的下一帧gF选择页面的主 / 顶框

# 使用搜索

快捷键操作快捷键操作
o打开 URL，书签或历史记录条目O在新标签页中打开 URL，书签或历史记录条目
b打开书签B在新标签页中打开书签
T在打开的标签页中搜索

快捷键操作快捷键操作
/进入查找模式
n向前循环到下一个查找匹配项N向后循环到上一个查找匹配项

# 浏览历史

快捷键操作快捷键操作
H回顾历史L前进的历史

# 操作标签

快捷键操作快捷键操作
t创建新标签
J，gT左移一个标签K，gt右移一个标签
^转到以前访问的标签
g0转到第一个标签g$转到最后一个标签
