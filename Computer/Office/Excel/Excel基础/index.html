# 单元格范围：

A1:A6 --> 冒号表示连续的单元格，即 A1 到 A6
A1,A6 --> 逗号表示离散的单元格，即 A1 和 A6

# 选择单元格

单个单元格：单击单元格
所有单元格：单击行号和列标左上角交叉处的 “全选” 按钮
整行：单击需选择行的行号
整列：单击需选择列的列标
不相邻的多个单元格：Ctrl

# 修改单元格

合并：开始 --- 合并后居中（选中再次单击取消合并）
插入：右击某个单元格或某行某列，快捷菜单 -- 插入
删除：选中单元格右击 -- 删除 ， Del 键删除内容

默认 3 个工作表，sheet1/2/3
右击工作表标签，快捷菜单中可以重命名、删除、插入、移动、复制 (建立副本)

# 录入数据

选中单元格就可以输入数据
双击单元格可以修改数据
Alt + Enter 强制换行...
Enter 键跳到下方单元格
Tab 键跳转到右方单元格
也可以上下左右键切换

单元格右下角的填充柄：往下拖动可复制、递增
Excel 输入数字默认去掉前面的 0
以及数字多了会以科学计数法表示 ---> 先输入英文单引号

delete 快速清空
ctrl+; 快速插入日期
ctrl+shift+; 快速插入日期

F4 切换公式引用
tab 自动补全函数名称
Alt + = 快速插入求和公式
Ctrl + ↓ 移动到最后一行
Ctrl + 1 打开数字格式窗口
Ctrl + R 向右填充
Ctrl + D 向下填充
ctrl + L 创建智能表 （ 插入 -- 表格）
Alt + Enter 单元格内强制换行
Ctrl + enter 选中区域批量输入相同内容
Ctrl + Shift + ↓ 选中当前位置到最后一行
文本左对齐，数字右对齐
通配符？和 * ，分别代表一个字符和任意个字符如果要精确匹配 * 或 ？ 这两个字符可以在 * ？ 前加上～

# 数据 — 数据有效性

数据有效性 — 条件允许 序列 — 来源输入 男，女

# 普通数据区域转换成智能表格

方法一：套用表格样式
开始 -- 套用表格样式菜单，然后选择任意一种样式

方法二：选中单元格区域，插入 -- 表格

方法三：ctrl + T

提示：拖动表格右下角的小三角可以拓展智能表格

快捷键功能备注
F4公式引用方式切换（$）4 种引用方式
Esc取消输入/
Alt+=快速求和公式标记
Ctrl+'显示 / 隐藏公式数字 1 旁边的符号
# 引用方式

相对引用：A1， B1
绝对引用：$A$1,$B$1
混合引用：A$1, $A1

# 公式

SUM () 求和
average () 平均值
Max () 最大值
Min () 最小值
Rank.eq (单元格，区域) 排名
IF (条件，"真","假") 条件判断
IF (条件，"真",IF (条件，"真","假")) IF 嵌套

Count （区域） 区域内包含数字的单元格个数
Countif （区域，条件） 区域内符合条件的单元格个数（条件计数）
Countifs（区域，条件，区域，条件） 区域内符合条件的单元格个数（多条件计数）
Counta （区域） 区域内非空的单元格个数

=AND () 与运算
= OR () 或运算
= NOT () 非运算
